US01 - Como usuário, quero fazer login no sistema para acessar minhas funcionalidades.

Critérios de Aceitação:
  O usuário deve informar e-mail e senha;
  O sistema deve validar as credenciais; 
  Em caso de sucesso, o usuário deve ser redirecionado para sua área;
  Em caso de erro, deve ser exibida mensagem de credenciais inválidas.

US02 - Como administrador, quero cadastrar turmas para organizar os estudantes por disciplinas e períodos.

Critérios de Aceitação:
  O administrador deve informar nome da turma e período;
  O sistema deve salvar a turma no banco de dados;
  A turma cadastrada deve aparecer na lista de turmas;
  Não deve permitir cadastro com campos obrigatórios vazios.

US03 - Como administrador, quero vincular estudantes às turmas para controlar sua participação nas aulas.

Critérios de Aceitação:
  O sistema deve listar estudantes cadastrados;
  O administrador deve selecionar turma e estudante;
  O vínculo deve ser salvo corretamente;
  O estudante deve aparecer na lista da turma.

US04 - Como professor, quero publicar atividades para que os estudantes possam acessá-las online.

Critérios de Aceitação:
  O professor deve informar título, descrição e prazo;
  A atividade deve ser vinculada a uma turma;
  Os estudantes da turma devem visualizar a atividade;
  O sistema deve registrar data de publicação.

US05 - Como estudante, quero enviar meus trabalhos para cumprir as atividades propostas pelo professor.

Critérios de Aceitação:
  O estudante deve selecionar uma atividade;
  O sistema deve permitir upload de arquivo;
  O envio deve ficar registrado com data e hora;
  O professor deve conseguir visualizar o arquivo enviado.

US06 - Como professor, quero registrar notas das avaliações para acompanhar o desempenho dos estudantes.

Critérios de Aceitação:
  O professor deve selecionar turma e atividade/avaliação;
  Deve ser possível inserir nota por estudante;
  As notas devem ser salvas no sistema;
  Os estudantes devem conseguir visualizar suas notas.

US07 - Como estudante, quero consultar minhas notas para acompanhar meu desempenho acadêmico.

Critérios de Aceitação:
  O sistema deve exibir todas as notas do estudante;
  As notas devem estar organizadas por disciplina;
  O estudante não deve visualizar notas de outros alunos;
  As informações devem estar sempre atualizadas.

US08 - Como estudante ou professor, quero trocar mensagens pelo sistema para facilitar a comunicação.

Critérios de Aceitação:
  O sistema deve permitir envio e recebimento de mensagens;
  As mensagens devem conter remetente, destinatário e data;
  O histórico da conversa deve ficar salvo;
  O usuário deve ser notificado sobre novas mensagens.

US09 - Como coordenador, quero acompanhar as turmas e atividades acadêmicas para monitorar o andamento das disciplinas.

Critérios de Aceitação:
  O coordenador deve conseguir visualizar as turmas cadastradas;
  O sistema deve permitir consultar informações gerais das turmas;
  O coordenador deve visualizar o andamento das atividades;
  O coordenador não deve alterar atividades ou notas dos estudantes.
